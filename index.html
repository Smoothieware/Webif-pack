Welcome

Welcome to your Smoothieboard's web interface. Here we list the different web interfaces designed by the Smoothie community. Please try them out and choose the one you prefer.

Classic

The classic web interface that used to be included in Smoothie's flash by default. Very minimalistic, a good base to start designing a custom interface.

Go

Imrahil's improved interface

Designed by Imrahil, an improved version of the classic interface, with more functionality, and much nicer looking. Translated in two languages, new translations welcome.

Go

Fabrica

An experimental simplified/feature-full interface, not usable yet, and shown here as a preview.

Go

Documentation

A static copy of the smoothieware.org documentation wiki, so you can access the documentation even without internet access.

Go

File explorer

A simple interface to explore the contents of the SD card through your browser.
