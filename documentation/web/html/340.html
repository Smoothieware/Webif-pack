The OSBR is now the TIM Review

You likely reached this page from a link to osbr.ca. As of September 2011, the Open Source Business Resource (OSBR) has been renamed the Technology Innovation Management Review (TIM Review). OSBR articles have been moved to this website and can be found in the issue archive.
